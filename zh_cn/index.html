ServerHub MVC

ServerHub 由一个 Node.js 框架和用于目录管理的辅助 CLI 工具组成。其中 ServerHub MVC (npm/serverhub-mvc) 是一个运行于 Node.js 的高效且简单的 MVC Web 服务器框架。

  • 轻巧敏捷
  • 稳定可靠
  • 开源并且可拓展

要开始了解和学习 ServerHub，请参阅 入门指南。如果需要阅读英文版，请访问 ServerHub.
