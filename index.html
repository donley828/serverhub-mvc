ServerHub MVC

ServerHub is a collection of both Nodejs framework and its CLI tool. And ServerHub MVC (npm/serverhub-mvc) is a fast and handy MVC web server framework that runs on Nodejs.

  • Lightweight and Fast
  • Stable and Reliable
  • Extendable and Open Source
